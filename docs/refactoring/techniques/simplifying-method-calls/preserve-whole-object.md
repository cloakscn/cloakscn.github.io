## 为什么重构？

问题是，每次调用你的方法之前，必须调用未来参数对象的方法。如果这些方法或方法获取的数据量发生变化，你将需要在程序中仔细找到十几个这样的位置，并在每个位置实现这些变化。

应用此重构技术后，获取所有必要数据的代码将存储在一个地方（方法本身）中。
