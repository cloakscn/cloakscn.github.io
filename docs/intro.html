    # 时光机

    2018 年我顺利考上了大学，虽然不是很好，不过也算是有人愿意收留，还不错。

    同年加入了学院实验室，开启了长达四年的求学之路。

    像博客中所写：迎着风，往前冲！

    这就是我的四年，很精彩，但也有遗憾~

    时间的车轮滚滚向前，裹挟着我走进烽台科技（北京）有限公司，工控安全行业的引领者。

    2022 一直在路上 ( •̀ ω •́ )✧

    一个普通的开发者，不是大牛但心向远方！
